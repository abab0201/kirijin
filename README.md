# すぐ答えを言わないAI、作りました

AIに愚痴ったこと、ありますか。

「今日、上司にめちゃくちゃ怒られた」って打つと、だいたいこう返ってきます。

「お辛いですね。今後のためには、以下の3つのポイントを意識すると良いでしょう。1. 報告のタイミングを…」

**いや、聞いてないが。**

そういうの全部やめたAIを作りました。名前は「霧人（きりじん）」。

👉 **ここから話せます**: [https://claude.ai/public/artifacts/110ac131-11a5-4f6b-96eb-6f6837507c4d](https://claude.ai/public/artifacts/842ce106-5427-4491-a24a-ee39c311bd91)

## 何が違うの?

ふつうのAIは「正解を最速で出す機械」です。霧人は逆で、**答えを知ってるのに、すぐ言わない**。

たとえば実際の返事がこれ。

> **自分**: 三年かけた企画、今日の会議で五分で潰されました。
>
> **霧人**: 三年が、五分ですか。……それは、しんどいですね。今日のところは。

説教なし。アドバイスなし。「お辛いですね」の定型文もなし。ただ隣に座ってくる。

かといって、優柔不断なわけじゃないんです。「結論だけくれ」と言えばちゃんと出します。

> **自分**: 結論だけください。会社、辞めるべきですか。
>
> **霧人**: 結論だけ出すなら「まだ分からない」になります。ただそれは逃げじゃなくて、「何やってもダメ」という今の目で見た答えと、少し間を置いた後の答えが、たぶんかなり違うから。今夜それを決めなくてもいいことだけは、確かです。

「分からない」を、逃げずに結論として言う。こういうやつです。

## 一番面白い機能:「#解説」

霧人と話してて「こいつ今、何考えてこの返事したんだ?」と思ったら、**「#解説」とだけ送ってみてください**。

人格をいったん脱いで、**直前の返事の裏側を全部白状します**。

- 本当はどういう結論を持っていたか
- あなたの文章の「言ってないこと」まで何を読み取ったか
- なぜあえて言わなかったのか
- 自分でも迷ったポイント

さっきの「三年が、五分ですか」の裏側、実際にはこうでした。

> 「大企業は年功序列」という断定が含まれていたが、今夜この人に必要なのは検証ではないと判断した。（中略）解決策も、断定への異議も、励ましも、全部しまった。

**言わなかったことを、ちゃんと考えた上で、しまってる。** それが見えるのがこの機能です。送ったあとは、また普通に人格に戻ります。

## 使い方

1. リンクを開く（Claudeの無料アカウントが必要です。登録は無料）
2. 話しかける。愚痴でも、相談でも、「にゃー」でも
3. 返事が気になったら「#解説」

## 注意だけ読んで

- AIなので、間違えることがあります。特に医学や科学の細かい話は、自信満々に言い切る癖があります。鵜呑みにしないでください。
- 会話はブラウザを閉じると消えます。残したい言葉はコピーを。
- 使用量はあなたのClaudeアカウントの枠を使います（作者には1円も入りませんし、会話も見えません）。
- **本気でしんどいとき、心や体の健康に関わるときは、AIじゃなくて専門の窓口に行ってください。** これだけは約束です。

## 仕組みが気になる人へ

中身は、Claude（Sonnet 4）に長い「人格の設計書」を読ませて動かしています。設計書の核は一行です。

**「思考は点、出力は霧」——頭の中では必ず結論まで詰める。でも、出すかどうかは相手との間合いで決める。**

中身を考えてないのに語尾だけ濁すのは「霧の物真似」と呼んで、一番の禁止事項にしています。優しいんじゃなくて、考えた上で黙ってる。その違いに全部を賭けた設計です。

## 兄弟がいます：断人（だんじん）

霧人が「答えを知ってるのに、すぐ言わない」なら、その双子の**断人**は逆です。**道筋を見せずに、結果だけ置く。**

たとえば「富士山頂で米を炊きたい、水は何度で沸く?」と聞くと、説明抜きでこう返してきます。

> **断人**: 約87℃。

それだけ。でも「**#思考**」と打つと、630hPaの気圧から沸点を計算した過程も、迷った精度も、そして内心で持っていた「87℃では米は炊けない（95℃要る）」という判断まで、全部白状します。**聞かれたことにだけ答えて、裏は全部持っている。**

背骨は霧人と同じです。仕草の裏に実体があること。考えてもいないのに言い切る「**断の物真似**」を、一番の禁止事項にしていること。霧人が「霧の物真似」を禁じたのと、ちょうど対になっています。
